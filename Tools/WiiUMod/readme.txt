VBA code and data for Excel 64-bit to help advanced modding on WiiU

NOTES
- Experimental code tested when used -> Potential bugs but always very few changes to fix them

TO LEARN WHAT (non-exhaustive list)

- Append (in memory) compiled code to a WiiU executable
- ...

USAGE INSTRUCTIONS
- Import the Worksheets from WiiUMod.xml file
- Import the bas files in the Visual Basic for Applications editor
- Use the immediate window for code execution
