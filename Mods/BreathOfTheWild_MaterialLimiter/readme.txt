Limits the materials total count

NOTES
- Don't enable Item Limiter graphic pack
- Max materials total count = user-defined
- Max material slots count = 160 by default (editable in patches.txt)
- Max materials count per slot = 999 by default (editable in patches.txt)
- If you (r)enable this graphic pack during a game, the materials total count limit = current materials total count + selected preset value
- If this graphic pack is enabled before starting a game, there are materials loss if the materials total count in the game save is greater than the selected preset value
- See readme.txt and rules.txt for additional information

INSTALLATION INSTRUCTIONS
- browse to your Cemu folder
- open the graphicPacks folder
- store the BreathOfTheWild_MaterialLimiter folder in this directory
- enable the Material Limiter graphic pack in Cemu
