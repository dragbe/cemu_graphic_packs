Disables stasis on non-hostile guardian stalkers

NOTES:
- Designed to fix this https://gamebanana.com/requests/13389 known bug
- Non-hostile = haven't seen Link yet
- Disables also stasis on guardian stalker legs
- Decayed guardians also affected by this mod

INSTALLATION INSTRUCTIONS
- browse to your Cemu folder
- open the graphicPacks folder
- store the BreathOfTheWild_GuardianStalkerI folder in this directory
- enable the Guardian Stalker I graphic pack in Cemu

https://gamebanana.com/gamefiles/12901
