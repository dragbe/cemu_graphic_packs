Limits both materials and foods

NOTES
- Data Layer mod (last version) required
- Active after (re)start/(re)load a new/existing game
- Depending on the selected presets, possible {materials/foods} loss
- To completely disable this mod, disable it in Cemu, save and (re)load the game save

INSTALLATION INSTRUCTIONS
- browse to your Cemu folder
- open the graphicPacks folder
- store the BreathOfTheWild_ItemLimiter folder in this directory
- enable the Item Limiter graphic pack in Cemu
