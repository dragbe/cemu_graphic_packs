Customizes the weather

NOTES:
- Don't enable the Weather graphics pack
- BreathOfTheWild_WeatherForecast.html web page to create new weather rules
- If you (r)enable the Weather Forecast graphic pack in an area with a constant weather (such as thundra plateau), you must re-enter the area so that the rules related to this weather work

INSTALLATION INSTRUCTIONS
- browse to your Cemu folder
- open the graphicPacks folder
- store the BreathOfTheWild_WeatherForecast folder in this directory
- enable the Weather Forecast graphic pack in Cemu
