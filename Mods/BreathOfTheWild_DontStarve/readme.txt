Eat, drink and sleep to survive

NOTES
- User defined amount of hearts auto lost per day
- User defined amount of stamina to consume to lost one 1/4 heart
- Active after the first pause in the game
- Sleep doesn't recover hearts
- To restore the beds default behaviour, comment the following lines in the patch_patches.asm file:
#0x021B7CDC = nop
#0x023a3358 = cmpw r0, r0

INSTALLATION INSTRUCTIONS
- browse to your Cemu folder
- open the graphicPacks folder
- store the BreathOfTheWild_DontStarve folder in this directory
- enable the Don't Starve graphic pack in Cemu
