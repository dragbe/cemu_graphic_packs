Materials not stored in the inventory and used for modded game mechanics (normal rupees still works)

PREREQUISITES
- Data Layer (last version)
- Durability UI (last version)

NOTES
[Ancient Crystal]
+ PutAncientCrystal
| +5 (-1*BombsNum.MinValue) bombs
| RupeeValue: 0xB08
| Dropped by guardians, Yiga footsoldiers, wooden boxes and 3 treasure chests
+ Remote bomb rune as bomb bag
| -1 bomb when bomb ready even if it not used afterwards
| Experimental option to disable the wind bomb
[Red Potion]
+ PutRedPotion
| restores 12 pieces of heart
| RupeeValue: 12
| Dropped by Yiga Blademasters, hinoxes, lynels, treasure Octoroks, barrels and pots
+ Healing potion system found in several Zelda games
[Master Key]
+ PutMasterKey
| Demo with 12 chests on the mainfield map containing the small keys required by 12 shrines
| RupeeValue: 0xB10
| SmallKey[0] game data = number of keys
+ Key ring system
| small key usable in any dungeon

INSTALLATION INSTRUCTIONS
[Mod System Core]
- browse to your Cemu folder
- open the graphicPacks folder
- store the BreathOfTheWild_FakeRupees folder in this directory
- enable the Fake Rupees graphic pack in Cemu
[Ancient Crystal]
- install BreathOfTheWild_AncientCrystal.bnp with BCML (pay attention to conflicts with other mods using custom drop tables)
- store the BreathOfTheWild_BombBag folder in the Cemu graphicPacks folder
- enable the Bomb Bag graphic pack in Cemu
[Red Potion]
- install BreathOfTheWild_RedPotion.bnp with BCML (pay attention to conflicts with other mods using custom drop tables)
[Master Key]
- install BreathOfTheWild_MasterKey.bnp with BCML
- store the BreathOfTheWild_KeyRing folder in the Cemu graphicPacks folder
- enable the Key Ring graphic pack in Cemu
