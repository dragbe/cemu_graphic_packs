Shares game data and features with other cemu patches

NOTES
- Don't disable this graphic pack
- Don't rename the graphic pack folder
- Allows to fix cemu patches compatibility with mods that modify game save (Second Wind for example)
- Allows (if needed) a better injection points management
- Allows (if needed) to share features

INSTALLATION INSTRUCTIONS
- browse to your Cemu folder
- open the graphicPacks folder
- store the !BreathOfTheWild_!DataLayer folder in this directory
- enable the Data Layer graphic pack in Cemu
