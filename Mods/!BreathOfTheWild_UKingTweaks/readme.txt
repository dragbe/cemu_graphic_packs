BOTW executable tweaks

NOTES
- Don't rename the graphic pack folder
- Various code optimizations
- New free codecaves in the executable (useful to port cemu patches on WiiU)
- Disables minor game features (new item notifications and more later)

INSTALLATION INSTRUCTIONS
- browse to your Cemu folder
- open the graphicPacks folder
- store the !BreathOfTheWild_UKingTweaks folder in this directory
- enable the UKing Tweaks graphic pack in Cemu
